# Kittygram

Социальная сеть для обмена фотографиями домашних кошек.  
Пользователи могут регистрироваться, добавлять питомцев, загружать их фото и смотреть карточки других кошек.

## Ссылка на проект

https://ваш-домен.ru/kittygram/

> Укажите рабочую ссылку на приложение.  
> Обычно это что-то вроде: `https://<IP-или-домен>/kittygram/`

## Технологии

**Backend:** Python 3.12, Django 5, Django REST Framework, Djoser, PostgreSQL, Gunicorn  
**Frontend:** React 17, React Router  
**Инфраструктура:** Docker, Docker Compose, Nginx, GitHub Actions, Docker Hub

## Функциональность

- регистрация и авторизация пользователей;
- добавление, редактирование и удаление карточек кошек;
- загрузка и отображение фотографий;
- просмотр списка всех питомцев.

## Запуск локально

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vviverlo/kittygram.git
   cd kittygram
2. Создайте файл .env на основе .env.example и заполните переменные.

3. Запустите контейнеры:
  docker compose up --build
  
4. Откройте в браузере: http://127.0.0.1:9000
